# 🛒 Grocery List App

Удобное приложение для планирования покупок. Проект позволяет добавлять новые пункты в список, отмечать выполненные задачи и полностью очищать список.

## 🚀 [Live Demo](https://IrinaSheshko.github.io/To-do-list-React/)

---

### 🌟 Основные функции:
* **Интерактивный список:** Добавление и удаление продуктов в реальном времени.
* **Стилизация:** Использование внешних изображений и CSS-анимаций для создания приятного интерфейса.
* **Состояние (State Management):** Управление массивом данных через React Hooks.
* **UX/UI:** Продуманная структура компонентов для легкой масштабируемости кода.

---

### 🛠 Стек технологий:
* **React.js**
* **JavaScript (ES6+)**
* **CSS3** (Flexbox, адаптивная верстка)
